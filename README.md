# next js 라우팅

- app -> page.js, dashboard(폴더)로 구성되어있는 경우 page.js는 "/", dashboard는 "/dashboard"로 라우팅
- 모든 라우팅은 app 폴더 안에 있어야한다.
- 라우팅이 매치되는 파일이름은 page.js(pages.tsx) 등 "page"라는 이름이어야 한다.
- 폴더 이름이 브라우저에서 보이는 URL의 path가 됩니다.
